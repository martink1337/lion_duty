# lion_duty
 
# Features

- Toggle duty on/off for whitelisted jobs
- That's it

# Requirements
- 1. ESX
- 2. MySQL

# Download & Installation

- Download https://github.com/martink1337/lion_duty.git
- Put it in the `resources` folder 

## Installation	
- Add this in your `server.cfg`:
```bash
start lion_duty
```
